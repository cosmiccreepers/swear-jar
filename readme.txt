Setup:

1. Install Python 3.5 (or above)
2. Open command prompt and navigate to the folder you downloaded this too
3. Run command "pip3 install -r requirements.txt"
4. After installation, run the script with "python swearjar.py"
5. Talk into the microphone, if a phrase contains a swear word, a notification will be received and tallied.
6. Exit the application by saying "Exit" or "Quit"
